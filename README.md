# Agro Inputs vs Commodities

## Project Overview
This project analyzes the relationship between agricultural input costs and commodity prices over a six-year period (January 2020 â€“ December 2025).

The goal is to understand how fluctuations in commodity prices and exchange rates impact the relative cost of agricultural inputs over time.

## Data Sources
- Daily commodity prices (soybean, corn, wheat, sorghum, sunflower) in ARS
- Daily agricultural input prices in USD
- Daily USD/ARS exchange rate

## Objectives
- Convert all prices to a common currency (USD)
- Analyze price trends over time
- Compare input costs against commodity prices
- Identify periods of favorable and unfavorable terms for producers

## Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
